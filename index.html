Why Blog?

置顶 | 发表于 2023-09-30 更新于 2023-10-02

我以前看到过一些经验分享的网站，顺着这些网站看到了很多前辈优秀的博客主页，大家通过博客分享着自己的见闻、感受、想法、知识和技术。正好赶上中秋十一假期，我有点感冒，身体不太好，窝在寝室，想着自己也搭建一个博客，书己所见，抒己所想。
有点累，先就写这么多吧。最近突发奇想，又想重读一本书，未来可能会开个专栏写写读书笔记啥的。

再读张晓风：第2记

发表于 2023-10-03 更新于 2023-10-19 分类于 专栏

6.《平视，也有美景》

什么是小人物呢？我们平时又是什么视角去看人呢？别人又是用什么视角看我们的呢？是不是小人物真的重要吗？如何去发现美景呢？

7.《我在》*

“我在，意思是说我出席了，在生命的大教堂里”，为什么要郑重地宣告？是因为面前真实的山川吗？是因为曾经走过和未来要走的岁月吗？是因为此时此刻真真切切地存在？无论是在物理上的真实可感，还是内心里那些丰富饱满的触动：为当年港胞隶属的牵挂，为能“紧急待命”的责任，为求水更比捐书急的同情，为“我在书中”的心绪万千？

所以作者，读者和我，都出席了，在生命的大教堂里。

8.《摇动过，但依然是我的土地》

9.《肉体有千万种受难的形态》

所以作家也可以是很好的医生，病人说不清楚的东西经过作家的笔墨说出，比如鲁迅的《狂人日记》、《孔乙己》、《祝福》，老舍的《骆驼祥子》。（这些文章其实只是有个大概印象，但我想对于这个命题的证明，我给出的材料是足够的了）其实我想问，要有怎样的履历才能看到这肉体背后千万种受难的形态呢？即使对于这千万种形态中的一二，窥见之后又是否能玩世不恭地轻描淡写一笔带过呢？

10.《陈年老茶》

但愿未来的我能放过现在写这些文字的自己，能放过最好。如果说不放过，那也算是常读常新了，至少有不放过的理由呢？他会像过去一样喜欢删掉那些自己不忍卒读的文字吗？至少我知道现在的自己不会删掉以前好玩的文字，我相信他也不会删掉这些推敲过的文字，这也算常读常新了。综上所述，给出一个不等式：
¥2000>$1000 (假设人生活在4维空间中，汇率8:1)，聪明的读者能猜到我想说的那个字吗？

注：

  1. 读书读少了，每次读书会情不自禁地认真，所以应该多读书才对。

再读张晓风：第1记

发表于 2023-10-02 分类于 专栏

1.《我喜欢》

真是的，这本书以喜欢来开始，开篇就有这么多喜欢，看来是不得不摘了。开篇是，”我喜欢生活，生命是…“，我只摘了后半句。粗浅的想，作者这个时候已经走过四五十年了吧，已经到了不在乎“人生的什么和什么”的阶段了。那生活对于我们的意义是什么呢，是一天天的过去，一点点的长进，从而能一步步的前进吗？

2.《种种可爱》*

嗯，可爱。
想吃熟透了的木瓜了，每次看到学校教育超市有卖木瓜，总心甘情愿被骗。

3.《关于拥抱》

4.《别人的同学会》*

考试不要作弊。

5.《仗美执言》

不知不觉间在冥思中成长，复在静定中一针一缕的完成固然可贵，但一开始就立好计划，打出旗号，拟定十年工作进度表，要把自己造成一位“现代化刺绣首饰制作人”当然也没有什么不好。颠倒一下似乎也不是作者否定的意思。自然之美难得，但刻意也未尝不美。只是作者为这样一种自然之美打动吗？从而愿意为她仗“美”执言。

注：

  1. 标题右边有*号的文章，是我印象里在另一本书里也有的文章，两本书前面的重复度似乎没有我想象中那么高…不过一次笔记里出现个一篇以前看过的还是没有问题的。所以有什么些新的收获呢？或者，重新有什么收获呢？
  2. 敲笔记好像也挺累人的，先写到这里吧。断断续续写了一段时间，再往下写，恐怕就要开始敷衍了。
  3. 好像写的速度跟不上看的速度，那就一天写一篇吧。
  4. markdown的排版还不是很熟练，有时间好好看一看。

再读张晓风：第0记

发表于 2023-10-01 更新于 2023-10-02 分类于 专栏

从这学期说起，俄罗斯文学的当代意义课上，教授跟我们聊作家、聊文学、聊人生、聊社会，讲普希金笔下的奥涅金，讲莱蒙托夫《诗人之死》的忧郁与海子的浪漫，讲果戈理早年的被赏识和晚年的怀疑，讲陈白尘的《升官图》首演受阻陈白尘骂走高官，演出台词一字不改，讲因为一枚象棋棋子使得工厂进口机器报废的《黑炮事件》，讲《假如我是真的》如何排练后又没能上演。实不相瞒，上课期间我一直在与实变作业缠斗，但教授所描述的这些饱满生动的人和事进了耳朵，也进了脑子。所以，这也激发了我讲述的欲望，留给读者，留给自己。
还有呢，更多的是，机缘巧合之下，这门课提醒我该停一停，多久没有读过一本书了？数学系里我的生活是以周期1（天）来重复的，上课、想问题、作业这三个过程往复，日复一日，也不说无聊。如果把这样的生活比作食堂4毛的白米饭的话，那读书就是4食堂里12块钱加了料的炒饭，让我短暂地觉得“肚子有货”，心满意足。
再读张晓风，我读过的书不多，心里有了读书的念头，也不想找新书读，于是总总原因下，萌生出重读张晓风的想法。小学的时候，我爸给我从地摊上买三字经和百家姓逼着我读和背，我读不下去，只记得“弟子规，圣人训。…”前几句，后来对我很好的语文老师向我们说起了台湾省的作家张晓风和她的散文集《那夜的烛光》，很巧的机会我把这本书买回来读了。只不过散文和小说不同，散文没有贯穿始终的主线剧情，当时的我应该体会到了一些意象的平静美好。再读张晓风，这让我想起以前做过的一篇阅读毕淑敏的《常读常新的人鱼公主》，每个阶段都会有不同的体会和思考。经过一些沉淀过后，我又能读出些什么不同呢？又在其中会不会有什么东西影响着我待人处事、观察世界的方式呢？我有一丝的期待，看看二十岁的自己能够读出些什么来，也希望能带给看这篇文章的人一点什么。（？究竟是什么呢？）
我在图书馆里借到的是《张晓风文集》，不过没关系，我应该还能想起来看过哪几篇，就当是重温那本《那夜的烛光》了。
就此码住，未来可能长文章不会太多，这段时间阅读笔记更新尽量保持1天1次。
